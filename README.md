# Tinta Scoop Bucket

[![Tests](https://github.com/oipoistar/scoop-bucket/actions/workflows/ci.yml/badge.svg)](https://github.com/oipoistar/scoop-bucket/actions/workflows/ci.yml) [![Excavator](https://github.com/oipoistar/scoop-bucket/actions/workflows/excavator.yml/badge.svg)](https://github.com/oipoistar/scoop-bucket/actions/workflows/excavator.yml)

[Scoop](https://scoop.sh) bucket for [Tinta](https://tinta.cc) — a fast, lightweight markdown viewer for Windows. Native C++ with Direct2D, single portable exe under 1 MB, no Electron. MIT licensed.

## Install

```pwsh
scoop bucket add tinta https://github.com/oipoistar/scoop-bucket
scoop install tinta
```

Updates ship automatically: `scoop update tinta`.

## What's in the bucket

| App | Description |
|---|---|
| [tinta](https://github.com/oipoistar/tinta) | Fast, lightweight markdown viewer for Windows (native Direct2D, <1 MB) |
